# 🍳🐱 Cookly - Compartilhe e Descubra Receitas

O Cookly é um site de culinária desenvolvido para amantes da gastronomia que desejam compartilhar suas receitas e explorar novas ideias para cozinhar.

Simples, intuitivo e funcional, o projeto foi criado com foco na praticidade e na experiência do usuário, tornando o processo de descobrir e publicar receitas algo rápido e agradável.


## 🎯 Objetivo

Criar uma plataforma acessível onde usuários possam:

- 📤 Compartilhar suas próprias receitas
- 🔍 Explorar novas ideias culinárias
- 📖 Organizar receitas de forma simples
- 🍽️ Incentivar a criatividade na cozinha
- 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

HTML5 → Estrutura da aplicação
CSS3 → Estilização e layout
JavaScript (Vanilla) → Interatividade e manipulação do DOM

- 💻 Tipo de Aplicação
- 🌐 Aplicação Web
- 🖥️ Foco em Desktop
- 🚀 Funcionalidades
- 📝 Cadastro e exibição de receitas
- 🔎 Visualização de diferentes pratos
- 📋 Interface simples e organizada
- ⚡ Interações dinâmicas com JavaScript


## 🎨 Design e Usabilidade

O Cookly foi pensado com foco em:

Interface limpa e intuitiva
Navegação simples
Organização visual das receitas
Experiência agradável para o usuário


## ⭐ Considerações Finais

O Cookly é um projeto simples, mas com grande potencial, mostrando como tecnologias básicas da web podem ser utilizadas para criar soluções úteis e agradáveis para o dia a dia.
